# Amazon Sales Dashboard (Tableau)

## 📊 Project Overview
This project presents an interactive Tableau dashboard analyzing sales performance and product profitability.

## 🔍 Key Insights
- Revenue trends over time
- Regional sales distribution
- Channel-wise contribution (Online vs Offline)
- Profitability analysis across product categories
- Identification of top and low performing products

## 🛠 Tools Used
- Tableau
- Excel (Data Source)

## 📁 Files Included
- Tableau Dashboard (.twbx)
- Dashboard Screenshots

## 📌 Features
- Interactive filters
- Insightful tooltips
- Clean and user-friendly layout
